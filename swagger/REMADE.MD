## swagger 安装
```
go install github.com/swaggo/swag/cmd/swag
```

### swagger 包依赖
```
github.com/swaggo/gin-swagger
github.com/swaggo/gin-swagger/files
```

### SWAGGER API生成
```
swag init
```
### 如果提示没有命令
> 将`GOPATH` bin 目录下的 `swag.exe` 复制到 `GOROOT` 的bin 目录下
